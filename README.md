# nuklear

Nuklear bindings for SDL3 renderer and SDL3 OpenGL3

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     nuklear:
       github: dipolito/nuklearcr
   ```

2. Run `shards install`
